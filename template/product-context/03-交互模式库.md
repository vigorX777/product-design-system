# 交互模式库

> 从实际需求中提炼的可复用交互模式。模板中仅保留结构说明和最小示例，不包含真实项目流程。

## 字段说明

| 字段 | 说明 |
|------|------|
| **编号** | IP-001 格式，递增 |
| **名称** | 简洁的模式名称 |
| **触发条件** | 什么情况下使用此模式 |
| **流程描述** | 步骤列表：用户操作 → 系统响应 |
| **状态说明** | 各状态的视觉表现 |
| **异常处理** | 错误/边界情况的处理方式 |
| **适用场景** | 适用的业务上下文 |
| **引用案例** | 引用的需求路径（必填） |

---

> **模板说明**：请在你的真实需求复盘中确认模式稳定后，再将模式追加到本文件中。不要把未验证的设想直接写入模式库。

## 示例条目（可替换）

## IP-001：示例交互模式

- **触发条件**：在此填写模式触发时机。
- **流程描述**：
  1. 用户操作：在此填写第一步用户动作。
  2. 系统响应：在此填写系统反馈。
  3. 用户操作：在此填写后续动作与系统联动。
- **状态说明**：
  - 默认态：在此填写默认视觉表现；
  - 执行态：在此填写处理中表现；
  - 完成态：在此填写完成后表现。
- **异常处理**：在此填写失败、空状态或边界情况处理方式。
- **适用场景**：在此填写适用页面、组件或业务上下文。
- **引用案例**：`vX.Y/产品线/需求名称/`
