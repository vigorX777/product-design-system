## 使用说明

本文件用于沉淀你的产品专属业务术语。每条术语包含以下字段：

| 字段 | 说明 |
|------|------|
| **术语** | 产品中使用的标准词汇 |
| **定义** | 简洁明确的含义说明（1-2 句话） |
| **所属模块** | 该术语归属的功能模块或业务域 |
| **相关术语** | 与之关联的其他术语 |
