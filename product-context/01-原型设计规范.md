# 原型设计规范

> 本文档定义 HTML 交互原型的技术路线和视觉规范，所有新建原型文件均须遵循。

## 技术路线

**结论**: 纯 CSS 手绘方案

**决策依据**:
cw-magic-box 是公司内网组件库，其页面资源（`lib.bundle.js`、样式表）全部通过 webpack 打包为相对路径私有 bundle，未发布至任何公共 CDN。经验证，`unpkg.com` 和 `jsdelivr.net` 均找不到 `bkui-vue2` 包（提示 "Package not found"）。因此，CDN 引入方案不可行，采用纯 CSS 手绘 MagicBox 视觉风格的方案，无外部依赖，单文件自闭合。

**调研记录**（2026-02-25）:
- `https://apps.uat-bkee5.canwaysoft.com/cw-magic-box/#/icon` 页面脚本均为相对路径（`./js/...`、`./example/static/lib.bundle.js`），无 CDN 引用
- `lib.bundle.js` 为 Vue 2.5.22 + bkui-vue2 合并打包产物，不可拆分单独引用
- `https://unpkg.com/bkui-vue2/` → "Package not found"
- `https://cdn.jsdelivr.net/npm/bkui-vue2/` → "Failed to fetch version info"
- GitHub `TencentBlueKing/bkui-vue2` 仓库 dist 目录未提供独立 UMD 构建文件

### 方案说明

使用 CSS 变量和纯 CSS 手绘 MagicBox 视觉风格，参考 `v4.5.x/监控策略目标显示无采集任务关联信息/drawer_replica.html` 的 CSS 结构（约 1500 行，无外部依赖）。

**核心 CSS 变量（MagicBox Design Token）**:

```css
:root {
  /* 品牌色 */
  --color-primary:  #1272FF;
  --color-success:  #27C274;
  --color-warning:  #FF9214;
  --color-danger:   #F43B2C;

  /* 文字色 */
  --color-text-primary:    rgba(0, 0, 0, 0.85);
  --color-text-secondary:  rgba(0, 0, 0, 0.45);
  --color-text-disabled:   rgba(0, 0, 0, 0.25);

  /* 边框 & 背景 */
  --color-border:      #D1D7E1;
  --color-bg-base:     #F5F7FA;
  --color-bg-white:    #FFFFFF;

  /* 尺寸 */
  --border-radius:     4px;
  --btn-height-normal: 32px;
  --font-size-base:    14px;

  /* 字体栈 */
  --font-family: -apple-system, BlinkMacSystemFont, 'PingFang SC', 'Microsoft YaHei', sans-serif;
}
```

**文件结构模板**:

```html
<!DOCTYPE html>
<html lang="zh-CN">

  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  </style>

  <script>
```

### 限制

- 不使用 webpack / vite / npm 等构建工具
- 不使用 Tailwind CSS 或 Ant Design 风格
- 单文件 HTML，内联 CSS + JS，无外部 CDN 依赖
- 视觉风格必须贴近 MagicBox（蓝鲸）设计系统，不使用 Bootstrap 或 Material 风格

#### 系统实测：操作结果 Toast

> 来源：2026-02-26 批量删除交互优化需求，用户提供系统截图后提炼。

**视觉特征**：
- **位置**：紧贴顶部导航栏底部，页面水平居中（不是右上角浮出）；
- **背景**：极浅绿色 `#EBF9EE`，绿色边框 `#A3D9B1`，轻阴影；

```css
  to   { opacity: 0; }
```

**JS 控制示例（3 秒自动消失）**：
```javascript
function showToast(message, duration = 3000) {
  const container = document.getElementById('toast-container');
  const toast = document.createElement('div');
  toast.className = 'toast is-entering';
  toast.innerHTML = `
    <span class="toast-icon">✓</span>
    <span class="toast-text">${message}</span>
    <span class="toast-close" onclick="this.parentElement.remove()">✕</span>
  `;
  container.appendChild(toast);
  setTimeout(() => {
    toast.classList.add('is-leaving');
    setTimeout(() => toast.remove(), 300);
  }, duration);
}
```

**尺寸速查**：顶部偏移与导航栏高度一致（通常 `52px`），图标直径 `20px`，padding `10px 16px`。

**JS（核心逻辑）**：
```javascript
// 展开/收起下拉，fixed 定位对齐
function toggleDropdown() {
  const ms = document.getElementById('multi-select');
  const dd = document.getElementById('ms-dropdown');
  const willOpen = !dd.classList.contains('is-open');
  ms.classList.toggle('is-open');
  dd.classList.toggle('is-open');
  if (willOpen) {
    const rect = ms.getBoundingClientRect();
    dd.style.top   = (rect.bottom + 4) + 'px';
    dd.style.left  = rect.left + 'px';
    dd.style.width = rect.width + 'px';
  }
}

// 勾选/取消选项
function toggleOption(el, e) {
  e.stopPropagation();
  const cb = el.querySelector('input[type="checkbox"]');
  cb.checked = !cb.checked;
  renderTags();
}

// 渲染已选 tag（锁定项无 × 按钮）
function renderTags() {
  const ms = document.getElementById('multi-select');
  ms.querySelectorAll('.ms-tag').forEach(t => t.remove());
  const selected = [];
  ms.querySelectorAll('#ms-dropdown .ms-option').forEach(opt => {
    const cb = opt.querySelector('input[type="checkbox"]');
    if (cb && cb.checked) {
      selected.push({ name: opt.textContent.trim(), locked: opt.classList.contains('is-disabled') });
    }
  });
  const placeholder = document.getElementById('ms-placeholder');
  if (selected.length > 0) {
    placeholder.style.display = 'none';
    selected.forEach(item => {
      const tag = document.createElement('span');
      tag.className = 'ms-tag';
      tag.innerHTML = item.name + (item.locked ? '' : ' <span class="ms-tag-close" onclick="removeTag(event,\'' + item.name + '\')">×</span>');
      ms.insertBefore(tag, placeholder);
    });
  } else {
    placeholder.style.display = '';
  }
}

// 点击 × 移除 tag
function removeTag(e, name) {
  e.stopPropagation();
  document.querySelectorAll('#ms-dropdown .ms-option:not(.is-disabled)').forEach(opt => {
    if (opt.textContent.trim() === name) opt.querySelector('input').checked = false;
  });
  renderTags();
}

// 点击下拉外部收起
document.addEventListener('click', function(e) {
  if (!e.target.closest('#multi-select')) {
    document.getElementById('ms-dropdown')?.classList.remove('is-open');
    document.getElementById('multi-select')?.classList.remove('is-open');
  }
});
```

**使用说明**：
- `#ms-dropdown` 建议放在 `<body>` 末尾（或至少在 Modal 外），避免堆叠上下文限制；
- 若 Modal 有多行编辑场景，需为每行下拉赋予唯一 ID，并在 `toggleDropdown` 中传参区分；
- 完整可运行示例见 `组件模板/多选弹窗.html`。

## 交互约定

- **按钮层次**：
  - 主操作（如提交、保存）使用**主要按钮**（Primary）。
  - 次要操作（如取消、返回）使用**次要按钮**（Default）。
  - 低频操作或表格内操作使用**文字按钮**（Text）。
- **状态反馈**：
  - 异步操作时，按钮应显示 loading 状态。
  - 不满足操作条件时，按钮应处于 disabled 状态，并可配合 Tooltip 提示原因。
  - 表单错误提示通常位于输入框下方或右侧，使用危险色（`#F43B2C`）。
- **弹窗规范**：
  - 宽度：`480px`（小）、`600px`（中，默认）、`800px`（大）。
  - 结构：标准的标题栏（Header）+ 内容区（Body）+ 底部操作区（Footer）三段式。

## CSS 命名空间隔离

将独立功能模块（如配置抽屉、浮层、弹窗）集成到综合原型时，**必须为该模块所有 CSS 类名加统一前缀**，与主页样式完全隔离。

**规则**:
- 每个独立模块选取一个简短前缀（如集群聚合规则抽屉用 `.agg-*`，告警规则弹窗用 `.alarm-*`）。
- 该模块内所有类名统一加前缀：`.agg-drawer`、`.agg-form`、`.agg-btn`、`.agg-table` 等。
- **禁止**在独立模块中直接使用主页通用类名（如 `.drawer`、`.form`、`.btn`、`.table`），即使视觉效果相同。
- 前缀选取原则：与模块功能相关、3~5 个字符、全小写连字符。

**适用场景**: 原型集成多个独立开发的功能模块，各模块 CSS 可能共享相同的类名（如 `.drawer`、`.row`、`.form-item`），隔离后可确保修改任一模块样式不影响其他模块。

**来源**: 需求1 集群聚合规则抽屉集成（使用 `.agg-*` 前缀），参见 `v4.5.x/观测中心/中广核项目需求/需求1：应用、服务层支持通过CMDB映射第三方系统的实例【中广核】/`。
