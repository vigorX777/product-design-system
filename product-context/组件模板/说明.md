# 组件模板使用说明

> 本目录包含符合原型设计规范视觉风格的 HTML 原型基础模板，可直接复制到新需求文件夹中使用。

## 可用模板

| 文件 | 适用场景 |
|------|----------|
| 标准列表页.html | 标准列表页（带搜索、筛选、分页） |

## 使用方式

1. 复制目标模板到新需求文件夹，重命名为 `{需求名称}.html`
2. 修改页面标题（`<title>` 和页头标题文字）
3. 按需调整列定义、表单字段、详情内容
4. 遵循 `product-context/01-原型设计规范.md` 中的视觉规范

## 视觉规范

参考 `product-context/01-原型设计规范.md` 获取完整的 CSS 变量和组件样式指南。

## 扩展模板

根据你的产品需要，可自行添加更多模板，例如：

| 文件（示例） | 适用场景 |
|------|----------|
| 弹窗表单.html | 弹窗表单（带表单验证占位） |
| 侧滑详情.html | 侧滑详情面板（带描述列表或 Tab） |
| dashboard.html | 数据看板（带图表占位区块） |
