# 业务术语表

> 嘉为鲸眼·监控中心产品的核心业务术语定义，确保需求文档的描述一致。

| 术语 | 定义 | 所属模块 | 相关术语 |
|------|------|----------|----------|
| 监控策略 | 定义监控目标的采集规则和告警规则的配置单元 | 监控配置 | 采集任务、监控目标 |
| 监控目标 | 被监控的对象实例，如主机、数据库、应用等 | 监控配置 | 监控策略、采集任务 |
| 采集任务 | 在监控目标上执行数据采集的任务配置 | 采集管理 | 采集插件、监控目标 |
| 采集插件 | 用于从监控目标采集指标数据的插件程序 | 采集管理 | 采集任务 |
| 动态分组 | 按规则动态匹配监控目标的分组方式 | 目标管理 | 动态拓扑、监控目标 |
| 动态拓扑 | 按拓扑关系动态关联监控目标的分组方式 | 目标管理 | 动态分组、监控目标 |
| 异常策略 | 定义监控目标出现异常时的处置规则 | 告警配置 | 监控策略、监控目标 |
| 拓扑分层 | 全景观测拓扑的垂直层级划分，包括终端、应用、服务、中间件、数据库、容器、操作系统、物理机、网络设备等分层 | 观测中心 | 对象模型、全景观测拓扑 |
| 对象模型 | 描述一类被监控实体的元数据定义，包含实例字段、关联关系字段及对应的 CMDB 模型映射 | 观测中心 | 拓扑分层、CMDB映射、关联关系字段 |
| CMDB映射 | 将对象模型与 CMDB 中的某一 CI 模型进行绑定，使系统可从 CMDB 同步实例数据 | 观测中心 | 对象模型 |
| 关联关系字段 | 对象模型配置中指定的一个 CMDB 关系类型字段，系统依此在拓扑图中绘制实例间连线 | 观测中心 | 对象模型、拓扑分层 |
| APM探针 | 通过应用性能管理（APM）技术自动发现的应用/服务实例数据来源，为系统内置对象模型，在拓扑分层中为锁定项 | 观测中心 | 对象模型、拓扑分层 |
| 听云服务 | 通过听云 APM 产品采集并经 CMDB 同步的「服务」类型实例，可关联到应用层或服务层拓扑分层 | 观测中心 | 对象模型、CMDB映射 |
| 听云实例 | 通过听云 APM 产品采集并经 CMDB 同步的「服务实例」类型实例，可关联到服务层拓扑分层 | 观测中心 | 对象模型、CMDB映射 |
| 全景观测拓扑 | 以分层拓扑图方式展示各层对象模型实例及其关联关系的可视化视图，是观测中心的核心功能页面 | 观测中心 | 拓扑分层、对象模型 |
| 拓扑分层管理 | 用于配置全景观测拓扑每个分层所关联对象模型的管理页面，当前版本配置全局生效 | 观测中心 | 拓扑分层、对象模型 |
| 采集节点 | 部署在网络中用于接收和处理 NetFlow/sFlow/IPFIX 等流量协议数据的采集组件 | 网络流量分析 | NetFlow、sFlow、IPFIX |
| NetFlow | 由思科开发的网络流量导出协议，用于收集 IP 网络流量信息，支持多厂商设备 | 网络流量分析 | 采集节点、sFlow、IPFIX |
| sFlow | 采样流量导出协议，支持多厂商网络设备的流量采集，与 NetFlow 为同类协议 | 网络流量分析 | 采集节点、NetFlow |
| IPFIX | 基于 NetFlow v9 的 IETF 标准流量导出协议，是 NetFlow 的标准化演进版本 | 网络流量分析 | 采集节点、NetFlow |
| 流量探索 | 基于时间范围、设备/接口、五元组等多维条件对网络流量数据进行检索和多维度分析的功能模块 | 网络流量分析 | 五元组、应用映射 |
| 应用映射 | 基于源/目的 IP、端口、协议（TCP/UDP）组合自定义应用识别规则，将网络流量归类到具体业务应用 | 网络流量分析 | 流量探索、五元组 |
| 五元组 | 唯一标识一条网络会话的五个要素：源IP、目的IP、源端口、目的端口、协议，是流量过滤和分析的基本单元 | 网络流量分析 | 流量探索、应用映射 |
| DSCP | 差分服务代码点（Differentiated Services Code Point），TCP/IP 包头中用于标记流量优先级（QoS）的字段 | 网络流量分析 | 流量探索 |
| 终端层 | 全景观测拓扑中位于最顶层的分层，展示终端侧（RUM）应用实例（如听云 Mobile 应用、听云 Web 应用、听云小程序应用），采用扁平节点展示，不做集群聚合，连线指向应用层节点 | 观测中心 | 拓扑分层、对象模型、RUM |
| 集群聚合规则 | 定义服务层、中间件层、数据库层中集群类型节点与实例类型节点聚合关系的规则配置，分为系统内置（APM 探针自动聚合）和手动配置（CMDB 来源需指定关联字段）两类 | 观测中心 | 对象模型、关联关系字段、拓扑分层 |
| 定开告警源插件 | 为第三方系统（如听云）定制开发的告警源接入插件，部署后可将听云节点的告警数据纳入鲸眼告警体系，使 CMDB 来源节点的告警展示逻辑与 APM 节点保持一致 | 观测中心 | 听云服务、全景观测拓扑 |

> 如有新术语，按表格格式追加，不修改已有条目。
